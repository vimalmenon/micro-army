# Helios — Admin Dashboard

**God of the Sun — illuminating what's hidden**

Helios (Ἥλιος) was the Titan god of the sun who drove his chariot across the sky each day, bringing light to the world. He saw everything from above — nothing escaped his gaze.

**Why Helios?**

This is the admin dashboard — a React single-page app that gives operators full visibility into the system. Messages, leads, and system status all visible at a glance. Like Helios illuminating the world, this dashboard shines a light on everything happening in the ecosystem.

**Domain:** Admin UI (messages, leads overview)
**Stack:** React + Vite + TypeScript
**Dependencies:** Hestia (backend API)
**Auth:** Authelia + API key

# Atlas — S3 Storage Service

**Titan who held up the Heavens**

Atlas (Ἄτλας) was a Titan condemned to hold up the sky for eternity — bearing the weight of the celestial sphere on his shoulders.

**Why Atlas?**

This microservice handles all file storage in S3. Like Atlas bearing the weight of the heavens, this service carries the burden of all binary data: images, audio, videos, documents, and backups.

**Domain:** File storage & retrieval
**Dependencies:** AWS S3 (`completeautomate-media` bucket)

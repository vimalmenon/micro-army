# Arachne — Tunnel Sync Service

**The Weaver (wrapped in irony)**

Arachne (Ἀράχνη) was a mortal weaver so skilled that she dared challenge Athena to a weaving contest. When Athena could find no fault in Arachne's work, she tore the tapestry and transformed Arachne into a spider, condemning her to weave for eternity.

**Why Arachne (the irony)?**

This microservice weaves together k3s ingress hostnames and Cloudflare tunnel configurations — keeping them in perfect sync. Like Arachne weaving intricate tapestries, this service weaves network config. And the twist? Cloudflare tunnels are *tunnels* — just as spiders live in tunnels (webs). Perfectly ironic.

**Domain:** Cloudflare tunnel configuration reconciliation
**Dependencies:** Cloudflare API, kubectl

# Azure Quiz — AZ-104 Practice App

**Test your Azure knowledge with timed quizzes.**

**Domain:** Azure certification (AZ-104) practice quiz
**Stack:** React (Vite), served as static build
**Deployment:** `ghcr.io/vimalmenon/azure.completeautomate`
**Endpoint:** `azure.completeautomate.com`

**What it does:**
- Timed AZ-104 practice quizzes
- Multiple-choice questions with instant feedback
- Score tracking and progress monitoring
- Designed for structured, balanced study sessions

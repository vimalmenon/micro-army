# Angelos — Messages Service

**Messenger Deity**

Angelos (Ἄγγελος) was a divine messenger in Greek mythology — the original root of the word "angel" itself. Angelos served as a herald between the divine and mortal realms, carrying news and information across boundaries.

**Why Angelos?**

This microservice powers the contact form on completeautomate.com — accepting messages from visitors and storing them for review. Like Angelos carrying messages between realms, this service carries messages from the public web into the internal system.

**Domain:** Contact form API (public endpoint)
**Dependencies:** Clio (DynamoDB)
**Endpoint:** POST /messages (public, CORS-enabled)
